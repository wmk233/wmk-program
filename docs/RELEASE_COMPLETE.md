## 使用方法

### 直接运行
1. 确保已安装 Python 3.7+
2. 运行启动脚本:
   - Windows: 双击 start.bat
   - MacOS: 运行 ./start.sh

### 开发运行
1. 安装依赖: pip install -r requirements.txt
2. 运行程序: python main.py
